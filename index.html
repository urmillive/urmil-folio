2017 – 2018 · rajkot → bca

A first laptop changed everything.

C++ conquered by class 12. Then BCA — and his own machine at last. Python all night: a voice assistant named Jarvis that answered back, and an invisibility cloak built with OpenCV. Code became magic.

2019 · the leap

A stranger paid him for code.

First freelance project at nineteen — and he never looked back. Three years of end-to-end client delivery: React, Angular, React Native, Next.js. Requirements to deployment, owned alone.

2020 – 2022 · the arena

Open source. Hackathons. A trophy.

Hacktoberfest from 2020. GDSC at Parul University, Google Cloud certified, an MCA in Cyber Security begun — and 12+ hackathon builds across IT domains, capped by triumph at Gateway Group’s TECHATHON.

2022 – 2025 · startups, then scale

From startup chaos to enterprise calm.

REST APIs and an online code editor at Fox Valley. Warelogg — a logistics startup built with IIT students. Then Asite: intern to Associate Software Engineer-I in 2½ years — the Angular & TypeScript era.

intern → engineer

aug 2025 · upsquare · now

Today he ships with AI at each elbow.

Software Engineer at Upsquare: 3–4 real products at once, scalable Angular + TypeScript systems, AI tools in every workflow. After hours: his own fleet of apps, live on the Play Store.
